# PLDialog

[![](https://jitpack.io/v/xm3366336/PLDialog.svg)](https://jitpack.io/#xm3366336/PLDialog)

Android开发过程中，Dialog必不可少的一个组件，本人常用的几个：
`提示框`、`进度框`、`输入框`、`选择框`、`看图片`、`输车牌`
